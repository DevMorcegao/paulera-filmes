# Brand Strategy

> This document defines the symbolic, visual and emotional identity of Paulera Filmes.
>
> It does **not** define UI components.
>
> Instead, it establishes the visual language that every future interface should inherit.

---

# Purpose

A visual identity is more than colors and typography.

It is the visual expression of an idea.

The purpose of this document is to define **how the project should be perceived before any content is read.**

When someone opens the website, they should immediately feel:

- Curiosity
- Calm
- Authenticity
- Cinema
- Nature
- Mystery

Everything described below exists to reinforce those emotions.

---

# Brand Essence

Paulera Filmes is a storyteller.

Not a videomaker.

Not a production company.

Not a content creator.

Stories are the product.

Cinema is the language.

Nature is the metaphor.

---

# Brand Statement

>

# Shape Language

Every interface should inherit shapes inspired by nature.

Prefer

Soft corners.

Organic curves.

Irregular compositions.

Flowing layouts.

Layered depth.

Avoid

Sharp mechanical geometry.

Rigid symmetry.

Industrial aesthetics.

Perfect grids dominating the experience.

---

Chrome.

Plastic.

Neon.

Heavy glassmorphism.

# Typography Philosophy

Typography should feel literary.

Not technological.

Headings should evoke books, cinema posters and editorial publications.

Body text should prioritize readability and rhythm.

Reading should feel comfortable even in long paragraphs.

Typography should never compete with imagery.

---

# Photography Direction

Images should prioritize emotion over perfection.

Natural lighting.

Real people.

Texture.

Movement.

Imperfections.

Backlight.

Golden hour.

Rain.

Smoke.

Dust.

Film stills.

Avoid

Stock photography.

Overprocessed images.

Artificial HDR.

Excessive sharpness.

---

# Video Direction

Videos should behave like cinema.

No autoplay with sound.

No distracting controls.

Transitions should feel intentional.

Videos are chapters.

Not decorations.

---

# Illustration

Illustration should be minimal.

Whenever possible, let nature communicate instead.

Illustrations should only appear if they reinforce storytelling.

---

# Iconography

Icons should feel subtle.

Thin.

Elegant.

Quiet.

Never playful.

Never cartoonish.

Never futuristic.

Icons should support navigation.

Never become visual protagonists.

---

# Texture Language

The interface should occasionally reveal subtle imperfections.

Paper grain.

Film grain.

Organic shadows.

Light leaks.

Natural depth.

Almost invisible.

Users should feel them more than consciously notice them.

---

# Motion Identity

Motion should resemble nature.

Leaves moving.

Branches breathing.

Light changing.

Fog drifting.

Nothing should accelerate suddenly.

Nothing should bounce excessively.

Nothing should feel robotic.

---

# Spatial Identity

Whitespace is part of the storytelling.

Large breathing spaces.

Generous margins.

Slow pacing.

Layouts should never feel crowded.

Visitors should feel invited to pause.

---

# Emotional Palette

Rather than assigning colors to components, assign emotions.

| Emotion | Visual Feeling |
|----------|----------------|
| Curiosity | Soft shadows, partially hidden elements |
| Trust | Stable compositions, grounded colors |
| Wonder | Volumetric light, subtle depth |
| Calm | Large spacing, slow transitions |
| Discovery | Progressive reveals |
| Reflection | Quiet typography, restrained contrast |

---

- Soft fog
- Sun rays

Not browsing a website.

Every interaction should reinforce this illusion.

# Decision Log

## D-007

### Decision

The tree will become the universal visual language of the project.

### Reason

Using a single symbolic system creates stronger memorability than combining unrelated visual references.

### Impact

Colors, spacing, motion, layouts and illustrations should all derive from this metaphor.

---

## D-008

### Decision

Natural materials become the primary visual inspiration.

### Reason

They reinforce authenticity and emotional warmth.

### Impact

Future UI components should avoid artificial finishes and technology-first aesthetics.

---

## D-009

### Decision

Typography should feel editorial rather than digital.

### Reason

Cinema and storytelling have stronger connections with publishing than with software interfaces.

### Impact

Future font selection should prioritize personality, readability and timelessness.

# AI Context

When generating interfaces for this project:

Never start from components.

Start from atmosphere.

Every layout should communicate emotion before functionality.

Every visual choice should reinforce the symbolic language established here.

If an element does not contribute to the Creative Tree metaphor, question whether it should exist at all.

The website should feel alive.

Not animated.

There is a difference.
