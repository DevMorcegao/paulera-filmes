# ADR-0003

Status

Accepted

---

# Title

Use the Tree as the primary navigation metaphor.

---

# Context

Traditional portfolios feel generic.

The project requires a memorable identity.

---

# Decision

The Tree becomes the central interaction model.

The homepage is built around one persistent living tree.

Users never move between different trees.

Instead, the virtual camera reveals different parts of the same tree.

Roots

↓

Trunk

↓

Branches

↓

Fruits

↓

Leaves

Scrolling represents camera movement.

Navigation is achieved through cinematic progression rather than page transitions.

The tree is constant.

The camera tells the story.

---

# Alternatives

Classic navbar.

Cards.

Timeline.

Sidebar.

---

# Consequences

Positive

Strong narrative.

Unique identity.

Memorable experience.

Negative

Higher implementation complexity.

More animation work.
