# ADR-0001

Status

Accepted

---

# Title

Use Next.js App Router as the application framework.

---

# Context

The project requires:

- Server Components
- Excellent SEO
- High performance
- Hybrid rendering
- Modern React architecture

---

# Decision

Use Next.js (App Router).

---

# Alternatives

React + Vite

Nuxt

Astro

Remix

---

# Consequences

Positive

Excellent SEO.

Streaming.

Server Components.

Image optimization.

Metadata API.

Negative

Learning curve.

App Router complexity.
