# Definition of Done

A feature is complete when:

✓ Matches specification

✓ Responsive across Desktop, Tablet and Mobile

✓ Accessible — keyboard navigation, screen reader labels, visible focus, WCAG AA contrast, touch targets ≥ 44×44px

✓ Reduced Motion respected

✓ Lighthouse >95

✓ Uses Design Tokens

✓ Uses Motion Tokens

✓ No TypeScript errors

✓ No lint warnings

✓ Tested

✓ Documentation updated

✓ Deploy Preview approved

✓ Visual review approved

Only then may it be merged.
